# Android-Bitcoin-Ticker
A Bitcoin price reporter that provides latest Bitcoin prices in popular currencies.
Implemented Concepts:
  - Dependency Management.
  - Root Permissions.
  - Network calls & JSON parsing.
  - API & REST.
